===============================
INSTRUCTIONS TO RUN THE PROGRAM
===============================
Pre-requisites: 
Have the following software installed in your system-
1. cmake (3.20 +)
2. make (4.0 +)
3. graphviz

Check if the following files are present:
1. avl.cpp
2. CMakeLists.txt

Carefully follow the steps to run the program:
1. Make a build directory.
2. Open a terminal an type the following commands
		
	$ cd build
	$ cmake ..
	$ make

3. Run the executable/binary file created in build directory.
